# 📊 Featured Project


## 📈 Sales Analytics Dashboard


### 📌 Project Overview

Developed an end-to-end **Sales Analytics Dashboard** using a Superstore dataset containing **9,000+ records**. Performed data cleaning, exploratory data analysis (EDA), and SQL-based analysis to uncover sales trends, profitability patterns, customer behavior, and regional performance. Designed an interactive Power BI dashboard with dynamic KPIs and visualizations to support data-driven business decisions.

### 🚀 Key Highlights

- 📊 Designed an interactive Power BI dashboard with dynamic KPIs
- 📈 Performed data cleaning and EDA using Python (Pandas)
- 🗄 Wrote SQL queries to analyze sales and profitability
- 🌍 Evaluated regional and category-wise performance
- 💰 Analyzed Discount vs Profit relationship
- 👥 Identified top-performing products and customer segments

### 🛠 Tools & Technologies

`Power BI` • `SQL` • `Python` • `Pandas` • `Excel`

### 📈 Business Impact

- Improved visibility into business performance through interactive dashboards.
- Identified profitable product categories and sales regions.
- Highlighted discount strategies affecting profitability.
- Delivered actionable insights for business decision-making.
